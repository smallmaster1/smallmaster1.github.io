# 浅谈智能车入门

# 为什么要参加智能车大赛

简介：

全国大学生智能汽车竞赛是由教育部高等教育司委托教育部高等学校自动化类专业教学指导委员会主办、先后由飞思卡尔半导体公司、恩智浦（中国）管理有限公司协办的赛事。竞赛过程包括理论设计、实际制作、整车调试、现场比赛等环节，要求学生组成团队，协同工作。该竞赛融科学性、趣味性和观赏性为一体，涵盖自动控制、模式识别、传感技术、电子、电气、计算机、机械与汽车等多学科专业的创意性比赛。

首先，谈一下我个人是如何一步步接触到智能车:

  1. 大学的迷茫，虽说入学就准备好要考研，但是毕竟对于刚入学的我来说比较遥远。对于空闲时间的安排，就是玩耍，主要是我耍也不会耍
  2. 学长、学姐，宣传，带到实验室参观了一番
  3. 开始购买相关资料，询问学长需要学什么
  4. 终于点亮了小灯...

其次，说一下实际用途:

  1. 含金量高，为你的保研、评优或是找工作的简历上添上漂亮的一段话

  2. 中国高等教育学会高校竞赛评估与管理体系研究工作组发布的全国普通高校大学生竞赛排行榜，榜上有名

    编辑

  3. 收获一段美好的回忆，包括和队友一起调车，熬夜到早晨去看日出...

# 你需要具备什么

  • 硬件方面，理论方面主要是模电和数电，实际动手方面需要会绘制电路板，进行焊接、调试，同时设计到相应的一些工具的使用，如万用表、烙铁、风枪、加热焊台、示波器等等。

  • 软件和硬件都包括的单片机
