# 初读印象

comment:: 运用了基于人工势场的导引方法规划了 AUV 的回坞路径，然后基于 AUV 的运动方程设计了滑模跟踪控制律来解决 AUV 在复杂的回收环境下的避障问题，并通过引入 “填平势场” 来引导 AUV 走出局部极小点。

# 文章骨架

# 创新点到底是什么？

novelty:: 引入 “填平势场” 来引导 AUV 走出局部极小点。

# 有什么意义？

significance:: 避免陷入局部极点，使得无法达到对接点。

# 使用了什么方法？

usage method:: #人工势场 + #滑模跟踪控制

# 优点和缺点？

  • 考虑了人工势场局部极值问题
  • 测试环境简单

# 机构 & 作者

who::[[西北工业 - 高剑]]

prospect:: 下一步将针对潜艇在运动状态下的 AUV 回收路径规划问题进行研究。
